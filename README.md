# Pet Phrase Plus

__在玩家发出的信息末尾加上一段文字，或者替换其中的某些文字！__  

---

## 配置文件  
配置文件位于`config/petphraseplus.properties`  
可以用`/petphraseplus`打开配置界面  
1.19.4也可以通过Modmenu模组打开配置界面

### mark  
mod的标识字符（只能是单个字符）

### replace  
用来替换的关键字  
示例：`我/咱`会将聊天信息中的所有“我”替换为“咱”  

### endInner  
后缀，放在标点符号内

### endOuter  
后缀，放在标点符号外

# Pet Phrase Plus

__在玩家发出的信息末尾加上一段文字，或者替换其中的某些文字！__  

---

## 配置文件  
配置文件位于`config/petphraseplus.json`  

### mark  
mod的标识字符（只能是单个字符）  
示例：`"mark": "&"`  

### replace  
用来替换的关键字  
示例：`"replace": [["我","咱"],["你","汝"]]`  

### endInner  
后缀，放在标点符号内  
示例：`"endInner": "喵~"`  

### endOuter  
后缀，放在标点符号外  
示例：`"endOuter: "————鲁迅"`
